📺 Two ways to use TradingView with MMDE:

Option 1 — Export CSV (quickest):
1. Open your TradingView chart
2. Right-click on the chart → Export Chart Data
3. Save the CSV → open it → copy all rows
4. Switch to the Paste CSV tab → paste → Parse & Load

Option 2 — Webhook (automatic, real-time):
TradingView sends candles to MMDE every time a candle closes.
Add this Pine Script to your chart:
//@version=5 indicator("MMDE Data Sender", overlay=true) // Settings mmde_url = input.string("https://your-mmde-url.onrender.com/api/tradingview/webhook/", "MMDE Webhook URL") secret = input.string("mmde-tv-secret-2026", "MMDE Secret") n_candles = input.int(30, "Candles to send", minval=5, maxval=100) // Build JSON with last N candles build_json() => result = '{"symbol":"' + syminfo.ticker + '",' result += '"interval":"' + timeframe.period + '",' result += '"market":"forex",' result += '"candles":[' for i = n_candles - 1 to 0 o = str.tostring(open[i], "#.#####") h = str.tostring(high[i], "#.#####") l = str.tostring(low[i], "#.#####") c = str.tostring(close[i], "#.#####") v = str.tostring(volume[i],"#") result += '{"open":' + o + ',"high":' + h result += ',"low":' + l + ',"close":' + c result += ',"volume":' + v + '}' if i > 0 result += ',' result += ']}' result // Alert on every bar close if barstate.isconfirmed alert(build_json(), alert.freq_once_per_bar_close)

To set up the webhook alert:
1. Add the script above to your chart
2. Click the alarm clock (Alerts) → Create Alert
3. Condition: MMDE Data Sender → Any alert() function call
4. Notifications: tick Webhook URL
5. Paste: https://your-mmde-url.onrender.com/api/tradingview/webhook/
6. Save → MMDE auto-analyses every new candle
